# Angular-Hyperbolic Laplace Lab

This project implements a new framework for analyzing Laplace transforms using:

- Angular representation (circular geometry)
- Hyperbolic representation
- Matrix-based boundary method (MBM)

## Features
- Interactive Streamlit app
- Analytical classification of functions
- Unified geometric interpretation
